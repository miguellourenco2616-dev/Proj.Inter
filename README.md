# Proj.Inter
Projeto Interdisciplinar


## Sistema de Logística 
## Descrição 
- Aplicativo de logística para controle de veículos e entregas após solicitação do pedido.

## Funcionalidades implementadas
- Cadastro de entregador e cliente
- Registro de entregas
- Atualização de status
- Banco de dados com tabelas e inserts

## Tecnologias
- Java
- SQL Server
- Umbrello
- HTML/CSS
- Canva
- GitHub
